## Getting Started

### Environment variables

The contact form sends emails through [Resend](https://resend.com). Copy the
example environment file and fill in your credentials before running the app:

```bash
cp .env.local.example .env.local
```

- `RESEND_API_KEY` — API key from your Resend dashboard.
- `RESEND_FROM_EMAIL` — verified email address that Resend can send from.
- `CONTACT_TO_EMAIL` — destination inbox for inbound contact requests.

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

Can start editing the page by modifying `app/page.tsx`. The page auto-updates as you edit the file.

This project uses [`next/font`](https://nextjs.org/docs/app/building-your-application/optimizing/fonts) to automatically optimize and load [Geist](https://vercel.com/font), a new font family for Vercel.
